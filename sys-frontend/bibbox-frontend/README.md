# bibbox-frontend

## Project setup
```
sudo apt-get update
sudo apt-get upgrade
sudo apt-get install build-essential
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.34.0/install.sh | bash
nvm install stable
nvm use stable
npm install -g @vue/cli
```

## Additional Installations
```
npm install axios
vue add bootstrap-vue
```
